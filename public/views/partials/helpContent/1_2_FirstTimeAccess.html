1.2  First Time Access

As per the 'real' ONEm service, users accessing ONEmSimulator for the first time will be automatically switched to the #ONEm menu, irrespective of the command sent. This situation happens in the following circumstances:

  • The user accessed the ONEmSimulator for the first time
  • The current user's session expired (session length is current set to 24 hours)
  • The ONEmSimulator server was restarted and/or the software version was updated
